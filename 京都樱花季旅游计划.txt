# 京都樱花季5日游行程规划

## 总览页面

**行程概要**
- **第一天**: 抵达京都 → 哲学之道 → 银阁寺 → 南禅寺
- **第二天**: 岚山一日游 (天龙寺 → 竹林小径 → 渡月桥 → 保津川游船)
- **第三天**: 清水寺周边 (清水寺 → 二年坂三年坂 → 八坂神社 → 祇园)
- **第四天**: 伏见稻荷大社 → 东福寺 → 三十三间堂
- **第五天**: 二条城 → 京都御苑 → 购物与返程

**预算**: 约75,000日元(含住宿)
**最佳樱花观赏期**: 4月1日-10日
**交通建议**: 购买ICOCA卡+岚山小火车票

## 每日详细行程

### 第一天: 哲学之道与银阁寺区域

**时间表**:
```
08:30 早餐 (推荐: 进进堂 银阁寺店)
09:30 哲学之道漫步 (最佳摄影点: 法然院附近)
11:00 银阁寺参观 (最佳摄影点: 银沙滩前)
12:30 午餐 (推荐: 顺正汤豆腐)
14:00 南禅寺 (最佳摄影点: 三门楼上)
16:00 永观堂 (樱花隧道)
18:00 晚餐 (推荐: 瓢亭别馆 需预订)
```

**地图**: [哲学之道区域地图链接]
**餐厅预订建议**: 瓢亭需提前2周预约

### 第二天: 岚山一日游

**时间表**:
```
07:30 早餐 (酒店内)
08:30 抵达岚山 (建议JR嵯峨野线)
09:00 天龙寺 (最佳摄影点: 曹源池庭院)
10:30 竹林小径 (清晨人少时最佳)
12:00 午餐 (推荐: 鲷匠HANANA)
13:30 保津川游船 (需提前购票)
15:30 渡月桥周边 (最佳摄影点: 桥北侧)
17:00 岚电樱花隧道 (夜间点灯)
18:30 晚餐 (推荐: 广川鳗鱼饭 需预订)
```

**地图**: [岚山区域详细地图]
**特别提示**: 小火车票建议提前1个月购买

### 第三天: 清水寺周边

**时间表**:
```
07:00 清晨清水寺 (开门即入避开人潮)
09:00 二年坂三年坂 (店铺刚开门时)
11:00 八坂神社 (舞殿前樱花)
12:30 午餐 (推荐: 菊乃井露庵 需预订)
14:00 祇园白川 (最佳摄影点: 辰巳大明神前)
16:00 建仁寺 (枯山水与樱花对比)
18:00 晚餐 (推荐: 十二段家寿喜烧)
```

**摄影提示**: 清水寺舞台需排队30-60分钟

### 第四天: 伏见稻荷与东福寺

**时间表**:
```
08:00 伏见稻荷大社 (清晨人少)
10:00 千本鸟居 (最佳摄影点: 奥社后方)
12:00 午餐 (推荐: 祢ざめ家鳗鱼饭)
13:30 东福寺通天桥 (樱花与枫叶对比)
15:30 三十三间堂 (1001尊观音像)
17:30 晚餐 (推荐: 天周天妇罗)
```

**交通提示**: 使用京阪电车一日券更划算

### 第五天: 二条城与京都御苑

**时间表**:
```
08:30 二条城 (开城即入)
10:30 京都御苑 (最佳摄影点: 近卫邸迹)
12:00 午餐 (推荐: 三嶋亭寿喜烧)
14:00 锦市场购物
16:00 新京极商店街
18:00 晚餐 (推荐: 先斗町的居酒屋)
```

**购物建议**: 可在BIC CAMERA免税购物

## 日语常用短语列表

1. すみません (Sumimasen) - 不好意思/打扰了
2. ありがとうございます (Arigatou gozaimasu) - 谢谢
3. これをください (Kore o kudasai) - 请给我这个
4. おいくらですか (O-ikura desu ka) - 多少钱?
5. トイレはどこですか (Toire wa doko desu ka) - 洗手间在哪里?
6. 写真を撮ってもいいですか (Shashin o totte mo ii desu ka) - 可以拍照吗?
7. 予約しています (Yoyaku shite imasu) - 我有预订
8. おすすめは何ですか (Osusume wa nan desu ka) - 有什么推荐?

## 交通卡使用建议

**ICOCA卡**:
- 购买地点: 关西机场或JR京都站
- 押金: 500日元(可退还)
- 充值: 车站自动售票机
- 使用范围: 巴士/地铁/便利店

**其他选择**:
1. 京都巴士一日券: 700日元(适合密集使用巴士)
2. 岚电一日券: 1200日元(含岚山区域交通)
3. 京阪电车一日券: 900日元(适合伏见稻荷方向)

**特别提示**:
- 岚山小火车需单独购票(620日元)
- 保津川游船约4100日元
- 出租车起步价约680日元(2km)
